## Say Hi to you!
``` python
import datetime

def sayHello(name):
  print((str(datetime.datetime.now())).split('.')[0] +\ 
    "\n" +\
    "{:}, hello!\n".format(name) +\
    "Welcome to 7V_SPACE.")
  
yourName = input("What's your name?")
sayHello(yourName)
``` 
